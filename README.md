## О проекте

**NONOIR Exchange** — статический многостраничный сайт, имитирующий работу сервиса по обмену криптовалюты на рубли. Проект выполнен в учебных целях: без фреймворков, без зависимостей, только нативный HTML, CSS и JavaScript.

Поддерживаемые направления обмена: **BTC, ETH, USDT (TRC-20 / ERC-20), LTC → RUB** и обратно.

---

## Возможности

- **Калькулятор курса** — мгновенный пересчёт суммы с учётом комиссии в обоих направлениях
- **Форма заявки** — генерирует уникальный номер (`NX-XXXXXX`) и показывает экран подтверждения
- **Тёмная / светлая тема** — переключается кнопкой, сохраняется в `localStorage`, синхронизируется с системными настройками (`prefers-color-scheme`)
- **Адаптивная вёрстка** — корректно отображается на мобильных устройствах (брейкпоинты 600 px и 900 px)
- **CSS-переменные** — единая система токенов для светлой и тёмной темы

## Курсы и комиссии

Задаются в файле `rates.js` и используются калькулятором на страницах обмена и курсов.

| Валюта | Курс (→ RUB) | Комиссия | Мин. сумма | Макс. сумма |
|--------|-------------|----------|------------|-------------|
| BTC | 7 980 000 ₽ | 1.2% | 0.001 BTC | 0.30 BTC |
| ETH | 284 000 ₽ | 1.0% | 0.02 ETH | 15 ETH |
| USDT | 94.20 ₽ | 0.8% | 100 USDT | 50 000 USDT |
| LTC | 8 150 ₽ | 1.1% | 1 LTC | 500 LTC |

Чтобы изменить курсы — отредактируйте объекты `RATES` и `COMMISSION` в `rates.js`.

## Запуск

Проект состоит из статических файлов — бэкенд не нужен.

**Вариант 1 — напрямую в браузере:**
```
Откройте index.html двойным кликом
```

**Вариант 2 — через локальный сервер (рекомендуется):**
```bash
# Python
python -m http.server 8080

# Node.js (npx)
npx serve .
```
Затем откройте `http://localhost:8080` в браузере.
